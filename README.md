# 🌍 Global COVID-19 Dashboard (Excel)

An interactive Microsoft Excel dashboard built using the **Our World in Data COVID-19 dataset**. This project provides insights into global COVID-19 cases, deaths, trends, and country-level performance through dynamic visualizations and interactive filters.

---

## 📌 Project Overview

This dashboard enables users to:
- 📊 Analyze global COVID-19 cases and deaths
- 🌎 Compare countries using interactive map charts
- 📈 Monitor trends in new cases over time
- 🎯 Filter data by country, continent, and year
- 📉 Identify the most affected countries

---

## ✨ Dashboard Features

- 📌 KPI Cards
  - Total Cases
  - Total Deaths
  - Number of Countries
  - Death Rate (%)
  - Survival Rate (%)

- 🌍 Filled Map Charts
  - Total Cases by Country
  - Total Deaths by Country

- 📊 Interactive Charts
  - Top 10 Countries by Cases
  - Top 10 Countries by Deaths
  - Daily New Cases Trend

- 🎛️ Interactive Slicers & Timeline

---
